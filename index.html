odatafy-mongodb

MongoDB Plugin for odatafy - convert oData requests to MongoDB queries. This npm module is part of the odatafy ecosystem. It leverages oDatafy Parser and the MongoDB aggregation framework to create MongoDb aggregation queries from oData URLs.
